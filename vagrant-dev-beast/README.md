Vagrant development box for web developers
==========================================

**Requirements**:
* VirtualBox 
* Vagrant (for Windows 1.1+)
* PuppetLabs Puppet

When you run command `vagrant up` 2 machines will be booted up with following name's and IPs.
* web (192.168.10.10)
* db (192.168.10.11)

**Pre-installed are**:
* PHP 5.4.13
* Oracle Java 7 
* NodeJS [TODO]
* Nginx 1.3
* MySQL 5.5
* Python
* Jetty 9 [TODO]
